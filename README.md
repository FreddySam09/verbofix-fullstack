# VERBOFIX
AI-driven web platform to assist people with stuttered speech using practice modules and progress tracking.

##  Repositories

###  **Backend – VerboFix API**
**GitHub:** https://github.com/FreddySam09/verbofix-backend  
A fast, scalable backend built with Django REST Framework, providing APIs for user authentication, speech-module data, progress tracking, and AI-powered assistance.

---

###  **Frontend – VerboFix Web Platform**
**GitHub:** https://github.com/Team-VerboFix/verbofix_website  
A responsive React + Tailwind web application featuring speech-practice modules, user dashboards, analytics, and interactive learning flows.
